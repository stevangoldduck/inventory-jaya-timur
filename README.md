### Implementation
copy `.env.example` , rename to `.env` -> setup your connection to DB
1. run `composer install`</br>
2. `npm install && npm run dev`</br>
3. `php artisan key:generate`</br>
4. `php artisan migrate`</br>
5. `php artisan passport:install`</br>
